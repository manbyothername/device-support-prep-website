# Exam Prep Web App

A web-based exam preparation tool focused on structured practice,
review, and performance tracking.

## Features
- Quiz modes (Balanced / Exam)
- Domain-based filtering
- Review incorrect answers
- Session statistics

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Supabase

## Status
Public preview. Features and content are subject to change.
